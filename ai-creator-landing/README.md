# AI Creator Landing

Готовый Vite + React проект для публикации на Vercel.

## Запуск локально
1. Установите Node.js
2. Откройте терминал в папке проекта
3. Выполните:
   npm install
   npm run dev

## Сборка
npm run build

## Публикация
Можно загрузить папку в GitHub и подключить репозиторий к Vercel.
